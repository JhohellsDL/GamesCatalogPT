# Reto Técnico - Recetas de cocina

## Descripción
Este proyecto ha sido desarrollado como parte de un reto técnico. Su propósito es permitir visualizar recetas de cocina.

## Tecnologías Utilizadas
- Kotlin
- Jetpack Compose
- Android Jetpack
- Room Database
- Retrofit
- Koin
- GoogleMaps API
- JUnit4
- Mockito

## Toma de Decisiones Tecnológicas
Para el desarrollo de este proyecto, se han seleccionado las siguientes tecnologías por las siguientes razones:

- **Kotlin**: Lenguaje moderno, seguro y recomendado para el desarrollo de aplicaciones Android.
- **Jetpack Compose**: Framework declarativo de UI que permite un desarrollo más rápido y eficiente.
- **Android Jetpack**: Conjunto de bibliotecas para seguir buenas prácticas en el desarrollo de Android.
- **Room Database**: ORM ligero y eficiente para la persistencia de datos.
- **Retrofit**: Biblioteca popular para la comunicación con APIs REST.
- **Koin**: Framework de inyección de dependencias simple y ligero, además es escalable para la migracion a multiplataforma.
- **GoogleMaps API**: Proporciona integración de mapas para mejorar la experiencia del usuario.
- **JUnit4**: Framework de pruebas unitarias ampliamente utilizado en el ecosistema de Android.
- **Mockito**: Biblioteca para la creación de mocks en pruebas unitarias.


## Instalación
1. Clona este repositorio:

   ```sh
   git clone https://github.com/JhohellsDL/ChallengeRecetasApp
   ```
2. Abre el proyecto en Android Studio.
3. Sincroniza las dependencias de Gradle.
4. Configura tu clave de API si es necesaria.
5. Ejecuta la aplicación en un emulador o dispositivo físico.

## Características
- Lista y buscador de las recetas (Home Screen)
- Tiene un elemento random en el Home
- Filtro por tipo de receta
- Detalles de la receta seleccionada (Detail Screen)
- Mapa geolocalización de origen de la receta (Map Screen)
- Pantalla de mapa, tiene ubicacion del usuario


## Arquitectura
Este proyecto sigue el patrón **MVVM (Model-View-ViewModel)** junto con **Clean Architecture** para mantener una estructura modular y escalable.
